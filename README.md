# Fabula Ultima Character Sheet v2

Scheda personaggio digitale per il GdR [Fabula Ultima](https://www.needgames.it/giochi/fabula-ultima/#core) di Needful Games.

Fabula Ultima è un marchio e un gioco di proprietà di Need Games. Questo progetto è un fan tool non ufficiale e non affiliato. Lo scopo è semplicemente offrire una versione più interattiva della [scheda ufficiale](https://fabula.nerdgrinder.com/wp-content/uploads/2024/02/Fabula-Ultima-Character-Sheet.pdf) da usare durante le nostre sessioni di gioco tra amici.

## Funzionalità
- Gestione attributi, status, risorse (HP/MP/IP)
- Classi, legami, orologi
- Tiro dadi animato
- Editor markdown per le note
- Multi-personaggio
- Import/export JSON
- Tema chiaro/scuro
- Salvataggio automatico

## Uso
Apri `index.html` nel browser. Nessuna installazione richiesta.
https://vampireneko.github.io/fabula-ultima-sheet-v2/
